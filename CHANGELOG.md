[1.0.0] - Initial Release

Added

- ELAN programming language core
- ".el" source file support
- ELAN compiler
- ELAN CLI tool
- Standard libraries
- Example programs

Libraries Included

- elancore
- elanmath
- elanstring
- elanfile
- elantime

CLI Commands

- "elan run"
- "elan compile"
- "elan install"
- "elan version"
