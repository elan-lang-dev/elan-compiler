Contributing to ELAN

Thank you for your interest in contributing to the ELAN Programming Language.

Owner: Elancheran C G

ELAN is an open-source programming language project designed for modern software development.

---

How to Contribute

1. Fork the repository.
2. Clone your fork to your computer.
3. Create a new branch for your changes.
4. Make your improvements or fixes.
5. Commit your changes.
6. Push the branch to your fork.
7. Open a Pull Request.

---

Development Areas

You can contribute to the following parts of ELAN:

- Compiler improvements
- Standard libraries
- Documentation
- Examples
- Bug fixes
- Performance improvements

Project folders:

src/        → Core compiler
cli/        → Command line interface
stdlib/     → Standard libraries
examples/   → Example programs
docs/       → Documentation

---

Coding Guidelines

Please follow these guidelines:

- Write clean and readable code
- Use clear function and variable names
- Add comments where necessary
- Keep changes small and focused

---

Reporting Bugs

If you find a bug:

1. Open the Issues section.
2. Describe the problem clearly.
3. Provide steps to reproduce the issue.

---

Feature Requests

If you have an idea for improving ELAN:

1. Create a new issue.
2. Explain the feature clearly.
3. Discuss with maintainers before implementing large changes.

---

Code of Conduct

Contributors should be respectful and collaborative when working on this project.

---

License

By contributing to ELAN, you agree that your contributions will be released under the project license.

---

Thank you for helping improve ELAN.
