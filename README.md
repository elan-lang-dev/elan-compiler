# ELAN Compiler 🚀

**ELAN — Build the Future**

The **ELAN Compiler** is the official compiler for the **ELAN Programming Language**.
ELAN is designed to be a **modern, simple, and powerful programming language** for developers worldwide.

File extension: **`.el`**

# What is ELAN?

ELAN is a programming language created to make development:

• Simple
• Fast
• Readable
• Powerful
• Scalable for large systems

It includes a **compiler, runtime, package manager, and IDE**.

hello.el

```

    print "Hello, World!"

}
```

Run program:

```
elan run hello.el
```

# Command Line Interface

Compile program

```
elan compile file.el
```

Run program

```
elan run file.el
```

Install library

```
elan install libraryname
```

Update libraries

```
elan update
```

Check version

```
elan version
```

---

# Compiler Architecture

The ELAN compiler works in several stages.

Source Code (.el)

↓

Lexer
(Tokenizes source code)

↓

Parser
(Builds Abstract Syntax Tree)

↓

Semantic Analyzer
(Check variables and functions)

↓

Optimizer
(Improves performance)

↓

Code Generator
(Generates executable code)

↓

Runtime Execution
